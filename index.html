Terms of Service & Privacy

1. Data Privacy

DocGenius AI operates primarily client-side. All documents, credentials, API configurations, and custom logos are stored directly in your local browser storage (LocalStorage). No document data is collected or stored on our servers.

2. Gemini API Connections

If you configure a custom Gemini API key, calls to the AI models will be routed directly from your browser to Google Gemini API servers. Your key is stored securely in LocalStorage and is never shared.

3. Usage Terms

This application is provided as a tool for document drafting and layout generation. Users are responsible for reviewing the accuracy and legality of all generated documents, certificates, and invoices.
